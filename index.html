Mathematik-Prüfung mit Lösungen

📏 Aufgabe 1: Lineare Gleichungssysteme (10 Punkte)

  1. Stelle die folgenden Gleichungen in die Form y = mx + b um:
    1. a) 2y - 2x = 2
    2. b) y + 1 = 3x
    3. c) 3y - 2x = 2
    4. d) 4y - 4x = 4

    Lösung:

    a) 2y - 2x = 2

    → 2y = 2x + 2 (addiere 2x auf beiden Seiten)

    → y = x + 1 (dividiere durch 2)

    b) y + 1 = 3x

    → y = 3x - 1 (subtrahiere 1 auf beiden Seiten)

    c) 3y - 2x = 2

    → 3y = 2x + 2 (addiere 2x)

    → y = (2/3)x + 2/3 (dividiere durch 3)

    d) 4y - 4x = 4

    → 4y = 4x + 4 (addiere 4x)

    → y = x + 1 (dividiere durch 4)

  2. Berechne die Anzahl der gemeinsamen Punkte der Geraden aus Teilaufgabe 1.

    Lösung:

    Geraden: a) y = x + 1, b) y = 3x - 1, c) y = (2/3)x + 2/3, d) y = x + 1

    - a) und d) sind identisch (gleiche Steigung m = 1, gleicher y-Achsenabschnitt b = 1) → unendlich viele gemeinsame Punkte.

    - Vergleiche a) mit b): x + 1 = 3x - 1

    → 2 = 2x → x = 1, y = 2 (ein Schnittpunkt: (1|2))

    - Vergleiche a) mit c): x + 1 = (2/3)x + 2/3

    → x - (2/3)x = 2/3 - 1 → (1/3)x = -1/3 → x = -1, y = 0 (ein Schnittpunkt: (-1|0))

    Ergebnis: a) und d) haben unendlich viele Punkte, mit b) und c) je einen Schnittpunkt.

  3. Löse das Lineare Gleichungssystem rechnerisch und bestimme die Lösungsmenge.

    Lösung (z. B. a) und b)):

    y = x + 1

    y = 3x - 1

    Gleichsetzen: x + 1 = 3x - 1

    → 2 = 2x → x = 1

    Einsetzen: y = 1 + 1 = 2

    Lösungsmenge: {(1|2)}

📈 Aufgabe 2: Nullstellen von Parabeln (10 Punkte)

Gegeben sind die Funktionen:

  • Parabel 1: y = x² + 2x
  • Parabel 2: y = x² + 2x - 1
  • Parabel 3: y = x² + 2x + 2
  1. Berechne die Nullstellen jeder Parabel mit der Mitternachtsformel.

    Mitternachtsformel: x = [-b ± √(b² - 4ac)] / (2a)

    1) y = x² + 2x (a = 1, b = 2, c = 0)

    → D = 2² - 4·1·0 = 4

    → x = [-2 ± √4] / 2 = [-2 ± 2] / 2

    → x₁ = 0, x₂ = -2

    2) y = x² + 2x - 1 (a = 1, b = 2, c = -1)

    → D = 2² - 4·1·(-1) = 4 + 4 = 8

    → x = [-2 ± √8] / 2 = [-2 ± 2√2] / 2 = -1 ± √2

    → x₁ ≈ -2.414, x₂ ≈ 0.414

    3) y = x² + 2x + 2 (a = 1, b = 2, c = 2)

    → D = 2² - 4·1·2 = 4 - 8 = -4

    → D < 0 → keine reellen Nullstellen

  2. Vergleiche die Diskriminante D = b² - 4ac mit Null und erkläre die Bedeutung des Ergebnisses.

    Lösung:

    1) D = 4 > 0 → 2 reelle Nullstellen

    2) D = 8 > 0 → 2 reelle Nullstellen

    3) D = -4 < 0 → keine reellen Nullstellen (Parabel liegt über der x-Achse)

  3. Zeichne die Graphen der Parabeln in ein Koordinatensystem.

    Wertetabelle für Parabel 1 (y = x² + 2x):

    x: -3 → y = 9 + (-6) = 3 | (-3|3)

    x: -2 → y = 4 + (-4) = 0 | (-2|0)

    x: 0 → y = 0 + 0 = 0 | (0|0)

    Scheitelpunkt: x = -b/(2a) = -2/2 = -1 → y = 1 - 2 = -1 | (-1|-1)

    Beschreibung: Öffnet nach oben, Nullstellen bei -2 und 0, Scheitel bei (-1|-1).

    (Ähnlich für die anderen Parabeln – zeichne sie mit diesen Punkten in ein Koordinatensystem!)

🔍 Aufgabe 3: Streckung und Stauchung in y-Richtung (10 Punkte)

Erstelle eine Wertetabelle für -4 ≤ x ≤ 4 und zeichne die Parabeln.

  1. p₁: y = ½x²
  2. p₂: y = -½x²

Wertetabelle p₁:

x = -4 → y = ½·16 = 8 | x = -2 → y = 2 | x = 0 → y = 0 | x = 2 → y = 2 | x = 4 → y = 8

Wertetabelle p₂:

x = -4 → y = -½·16 = -8 | x = -2 → y = -2 | x = 0 → y = 0 | x = 2 → y = -2 | x = 4 → y = -8

Beschreibung: p₁ öffnet nach oben, gestaucht (flacher als y = x²), p₂ öffnet nach unten, ebenfalls gestaucht.

Zusatzfrage: Wie verändert sich die Form der Parabel, wenn der Faktor vor x² größer oder kleiner wird?

Lösung: Ein größerer Faktor (z. B. 2) macht die Parabel steiler, ein kleinerer (z. B. ½) flacher. Negativer Faktor spiegelt sie an der x-Achse.

📍 Aufgabe 4: Scheitelpunkt einer Parabel und Parabelgleichungen (10 Punkte)

  1. Bestimme den Scheitelpunkt der folgenden Parabeln durch quadratische Ergänzung:
    1. a) y = x² + 8x + 17
    2. b) y = x² - 20x + 50
    3. c) y = x² - 7x

    a) y = x² + 8x + 17

    → y = (x² + 8x + 16) + 17 - 16 (Ergänzung: (8/2)² = 16)

    → y = (x + 4)² + 1 → Scheitelpunkt: (-4|1)

    b) y = x² - 20x + 50

    → y = (x² - 20x + 100) + 50 - 100 (Ergänzung: (-20/2)² = 100)

    → y = (x - 10)² - 50 → Scheitelpunkt: (10|-50)

    c) y = x² - 7x

    → y = (x² - 7x + 12.25) - 12.25 (Ergänzung: (-7/2)² = 12.25)

    → y = (x - 3.5)² - 12.25 → Scheitelpunkt: (3.5|-12.25)

  2. Bestimme die allgemeine Parabelgleichung der Form y = x² + bx + c, wenn der Scheitelpunkt gegeben ist:
    1. a) S(-2 | 1)
    2. b) S(-3 | -5)
    3. c) S(4 | -5)

    Scheitelpunktform: y = a(x - h)² + k, hier a = 1

    a) S(-2|1) → y = (x + 2)² + 1 = x² + 4x + 4 + 1 = x² + 4x + 5

    b) S(-3|-5) → y = (x + 3)² - 5 = x² + 6x + 9 - 5 = x² + 6x + 4

    c) S(4|-5) → y = (x - 4)² - 5 = x² - 8x + 16 - 5 = x² - 8x + 11

🔗 Aufgabe 5: Zuordnung von Parabelgleichungen (10 Punkte)

Allgemeine Form:

  1. p₁: y = x² - 2x - 4
  2. p₂: y = x² + 2x - 4
  3. p₃: y = x² + 4x + 1
  4. p₄: y = x² - 4x + 1

Scheitelpunktform:

  1. q: y = (x - 2)² + 4
  2. r: y = (x + 2)² + 3
  3. s: y = (x - 1)² - 5
  4. t: y = (x + 2)² - 3

Lösung:

p₁: y = x² - 2x - 4 = (x - 1)² - 1 - 4 = (x - 1)² - 5 → passt zu s

p₂: y = x² + 2x - 4 = (x + 1)² - 1 - 4 = (x + 1)² - 5 → kein Match

p₃: y = x² + 4x + 1 = (x + 2)² - 4 + 1 = (x + 2)² - 3 → passt zu t

p₄: y = x² - 4x + 1 = (x - 2)² - 4 + 1 = (x - 2)² - 3 → kein Match

Zuordnung: p₁ → s, p₃ → t (andere haben keine direkte Entsprechung).
